本页记录 `asr-data` 0.1.0 暴露的 Python 接口。所有类型均可直接从
`asr_data` 导入。

## Audio {id="audio"}

```python
Audio.from_file(path: str, id: str | None = None) -> Audio
Audio.from_url(url: str, id: str | None = None) -> Audio
Audio.from_bytes(data: bytes, id: str | None = None) -> Audio
Audio.from_pcm(
    data: bytes,
    sample_rate: int,
    channels: int = 1,
    id: str | None = None,
) -> Audio
```

属性：`id`、`duration_ms`、`timelines`、`metadata`。

```python
audio.timeline(channel: str | int) -> Timeline | None
audio.ensure_timeline(channel: str | int) -> Timeline
audio.remove_timeline(channel: str | int) -> bool
audio.load() -> Waveform
await audio.aload() -> Waveform
audio.validate() -> None
```

## Waveform {id="waveform"}

```python
Waveform(samples, sample_rate: int, channels: int = 1)
```

只读属性：`sample_rate`、`channels`、`is_normalized`、`frame_count`、
`duration_ms`、`source_format`。

```python
waveform.numpy()
waveform.display(autoplay: bool = False) -> None
waveform.to_mono() -> Waveform
waveform.channel(index: int) -> Waveform
waveform.resample(sample_rate: int) -> Waveform
waveform.normalize() -> Waveform
waveform.slice_ms(start_ms: int, end_ms: int) -> Waveform
```

## Timeline {id="timeline"}

属性：`id`、`audio_id`、`duration_ms`、`annotations`。

```python
timeline.add_speech(start_ms, end_ms, source="vad", confidence=None,
                    source_kind="stage") -> Annotation
timeline.add_silence(start_ms, end_ms, source="vad", confidence=None,
                     source_kind="stage") -> Annotation
timeline.add_transcription(start_ms, end_ms, text, source="asr",
                           language=None, confidence=None, status="final",
                           source_kind="stage") -> Annotation
timeline.add_speaker(start_ms, end_ms, speaker, source="diarization",
                     confidence=None, status="final",
                     source_kind="stage") -> Annotation
timeline.annotations_by_source(source, source_kind="model") -> list[Annotation]
timeline.transcript_by_source(source, source_kind="model") -> Transcript
timeline.remove_annotations_by_source(source, source_kind="model") -> int
timeline.relabel_annotations_source(from_source, to_source,
                                    from_source_kind="stage",
                                    to_source_kind="model") -> int
timeline.transcript() -> Transcript
```

## Annotation {id="annotation"}

只读属性：`id`、`start_ms`、`end_ms`、`status`、`kind`、`text`、
`confidence`、`source_kind`、`source`、`speaker`、`language`。

`Transcript` 提供 `text` 和 `language` 属性。

`AudioFormat` 提供 `encoding`、`sample_rate` 和 `channels` 属性。

## AudioDB {id="audiodb"}

```python
AudioDB(path: str, read_only: bool = False)

db.insert(audio: Audio) -> None
db.query(limit=100, *, after=None, min_duration_ms=None,
         max_duration_ms=None, metadata=None) -> list[Audio]
db.update(audio: Audio) -> bool
db.update_many(audios: list[Audio]) -> int
db.delete(audio_id: str) -> bool
db.set_metadata(key: str, value) -> None
db.metadata_value(key: str)
db.delete_metadata(key: str) -> bool
```

此外支持 `db[audio_id]`、`audio_id in db`、`len(db)`、迭代以及只读属性
`db.metadata`。

## 类型别名 {id="types"}

```python
AnnotationStatus = Literal["partial", "final", "revised", "deleted"]
AnnotationSourceKind = Literal["user", "model", "stage", "system"]
AnnotationKind = Literal[
    "speech", "silence", "token", "transcription", "sentence",
    "speaker", "language", "hotword", "acoustic_event", "diagnostic",
]
```

底层错误统一以 `AsrDataError` 暴露；按 ID 读取不存在的记录时使用 Python
标准 `KeyError`。
