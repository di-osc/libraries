## AudioSource {id="audio-source",tag="class"}

描述尚未解码的音频来源。来源可以是文件路径、URL、编码字节、base64 字符串或 PCM
S16LE 字节。`AudioSource` 可以加载为完整 [`Audio`](#audio)，也可以流式产生
[`AudioChunk`](#audio-chunk)。

## AudioSource.from_path {id="audio-source-from-path",tag="class method"}

从本地音频文件创建来源。文件会在调用 `load()` 或 `stream()` 时解码。

> #### Example
>
> ```python
> from pathlib import Path
> from tempfile import TemporaryDirectory
> from urllib.request import urlretrieve
>
> from asr_data import AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
>
> with TemporaryDirectory() as directory:
>     path = Path(directory) / "example.wav"
>     urlretrieve(URL, path)
>     source = AudioSource.from_path(str(path))
>     audio = source.load()
>     print(audio.duration_ms)
> ```

| Name        | Description                   |
| ----------- | ----------------------------- |
| `path`      | 本地音频文件路径。~~str~~     |
| **RETURNS** | 新的音频来源。~~AudioSource~~ |

## AudioSource.from_url {id="audio-source-from-url",tag="class method"}

从 HTTP 或 HTTPS 地址创建来源。网络请求在加载或流式读取时执行。

> #### Example
>
> ```python
> from asr_data import AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> source = AudioSource.from_url(URL)
> audio = source.load(mono=True, sample_rate=16_000)
> print(audio.sample_rate, audio.channels, audio.duration_ms)
> ```

| Name        | Description                     |
| ----------- | ------------------------------- |
| `url`       | HTTP 或 HTTPS 音频地址。~~str~~ |
| **RETURNS** | 新的音频来源。~~AudioSource~~   |

## AudioSource.from_pcm {id="audio-source-from-pcm",tag="class method"}

从 PCM S16LE 字节创建来源。

> #### Example
>
> ```python
> import numpy as np
>
> from asr_data import Audio, AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> audio = Audio.from_url(URL).to_mono().resample(16_000)
> pcm_bytes = (
>     np.clip(audio.samples, -1.0, 1.0) * 32_767
> ).astype("<i2").tobytes()
>
> source = AudioSource.from_pcm(
>     pcm_bytes,
>     sample_rate=16_000,
>     channels=1,
> )
> decoded = source.load()
> print(decoded.sample_rate, decoded.channels)
> ```

| Name          | Description                   |
| ------------- | ----------------------------- |
| `data`        | PCM S16LE 字节。~~bytes~~     |
| `sample_rate` | 每秒采样数。~~int~~           |
| `channels`    | 声道数，默认为 `1`。~~int~~   |
| **RETURNS**   | 新的音频来源。~~AudioSource~~ |

## AudioSource.load {id="audio-source-load",tag="method"}

同步解码完整音频，可在解码时完成重采样和单声道混合。

> #### Example
>
> ```python
> from asr_data import AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> source = AudioSource.from_url(URL)
> audio = source.load(sample_rate=16_000, mono=True)
> print(audio.duration_ms)
> ```

| Name           | Description                                       |
| -------------- | ------------------------------------------------- |
| _keyword-only_ |                                                   |
| `sample_rate`  | 目标采样率；省略时保留原采样率。~~Optional[int]~~ |
| `mono`         | 是否下混为单声道。~~Optional[bool]~~              |
| **RETURNS**    | 解码后的完整音频。~~Audio~~                       |

异步程序使用相同参数调用 `await source.aload(...)`。

## AudioSource.stream {id="audio-source-stream",tag="method"}

按固定目标时长逐块解码音频。

> #### Example
>
> ```python
> from asr_data import AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> source = AudioSource.from_url(URL)
>
> for chunk in source.stream(chunk_size_ms=100):
>     print(chunk.offset_ms, chunk.duration_ms, chunk.is_final)
> ```

| Name            | Description                                        |
| --------------- | -------------------------------------------------- |
| `chunk_size_ms` | 每块的目标时长，默认为 `100`。~~int~~              |
| _keyword-only_  |                                                    |
| `sample_rate`   | 可选目标采样率。~~Optional[int]~~                  |
| `mono`          | 是否下混为单声道。~~Optional[bool]~~               |
| **YIELDS**      | 带偏移和结束标志的音频块。~~Iterator[AudioChunk]~~ |

异步迭代使用 `source.astream(...)`，参数与返回元素相同。

## Audio {id="audio",tag="class"}

解码后的 `float32` 音频样本。`Audio` 是内存中的波形，不是数据库记录。

> from asr_data import Audio, AudioSource

> audio = Audio.from_source(source)
> print(audio.sample_rate, audio.channels)
> ```

| Name        | Description                 |
| ----------- | --------------------------- |
| **RETURNS** | 解码后的完整音频。~~Audio~~ |

异步环境使用 `await Audio.aload_from_path(path)` 或
`await Audio.aload_from_source(source)`；两者同样返回 [`Audio`](#audio)。

## Timeline {id="timeline",tag="class"}

某个声道上的时间轴。它将参考真值与模型预测分成两个独立集合。

| Name          | Description                                         |
| ------------- | --------------------------------------------------- |
| `id`          | Timeline 唯一 ID。~~str~~                           |
| `audio_id`    | 所属 AudioDoc ID，可修改。~~str~~                   |
| `duration_ms` | Timeline 总时长。~~int~~                            |
| `reference`   | 不带 source 的参考标注。~~ReferenceAnnotations~~    |
| `prediction`  | 必须带 source 的预测标注。~~PredictionAnnotations~~ |

容完全相同的重复添加会返回已有 annotation，不会插入第二条记录。

## Timeline.eval {id="timeline-eval",tag="method"}

使用 reference 评估指定来源的 prediction。转写与 Speech 可以来自不同模型，也可以
只评测其中一项。转写默认先执行内嵌中文 TN，再计算 CER。

> #### Example
>
> ```python
> from asr_data import AudioDoc, AudioSource
> from asr_data.annotation import Transcription
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> doc = AudioDoc(AudioSource.from_url(URL))
> timeline = doc.ensure_timeline("mono", duration_ms=4_204)
> text = "甚至出现交易几乎停滞的情况"
>
> timeline.reference.add_speech(0, timeline.duration_ms)
> timeline.reference.add_transcription(
>     0, timeline.duration_ms, Transcription(text, language="zh")
> )
> timeline.prediction.add_speech(
>     0, timeline.duration_ms, source="silero-vad"
> )
> timeline.prediction.add_transcription(
>     0,
>     timeline.duration_ms,
>     Transcription(text, language="zh"),
>     source="qwen-asr",
> )
>
> result = timeline.eval(
>     transcription="qwen-asr",
>     speech="silero-vad",
> )
> print(result.transcription.cer, result.speech.f1)
> ```

| Name            | Description                                        |
| --------------- | -------------------------------------------------- |
| _keyword-only_  |                                                    |
| `transcription` | 要评测的转写 prediction 来源。~~Optional[str]~~    |
| `speech`        | 要评测的 Speech prediction 来源。~~Optional[str]~~ |
| `normalize`     | 是否执行内嵌中文 TN，默认为 `True`。~~bool~~       |
| **RETURNS**     | 本次请求的评测结果。~~TimelineEvaluation~~         |

至少需要提供 `transcription` 或 `speech`。对应的 reference 或指定来源 prediction
不存在时抛出 [`AsrDataError`](#asr-data-error)。

## ReferenceAnnotations.add_speech {id="reference-add-speech",tag="method"}

添加一段语音活动范围。

> #### Example
>
> ```python
> from asr_data import AudioDoc, AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> source = AudioSource.from_url(URL)
> audio = source.load()
> timeline = AudioDoc(source).ensure_timeline("mono", audio.duration_ms)
>
> speech = timeline.reference.add_speech(
>     0,
>     timeline.duration_ms,

## AnnotationKind {id="annotation-kind",tag="type alias"}

描述 annotation payload 的类型。

| Value              | Description      |
| ------------------ | ---------------- |
| `"speech"`         | 语音活动范围。   |

## AnnotationPayload {id="annotation-payload-alias",tag="type alias"}

`Annotation.payload` 的返回类型。Python 公开的主要 payload 类型统一位于
`asr_data.annotation` 模块。

```python
AnnotationPayload = Speaker | Token | Transcription | dict[str, object] | None
```

`Speech` 的 payload 为 `None`。`Sentence`、`Language` 和 `AcousticEvent` 暂无
Python payload 类，从数据库或 Rust 数据读取时以字典形式返回。

## AsrDataError {id="asr-data-error",tag="exception"}

`asr-data` 数据校验、音频处理和数据库操作使用的公共异常类型。

> #### Example
>
> ```python
> from asr_data import AsrDataError, AudioDoc, AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> doc = AudioDoc(AudioSource.from_url(URL))
>
> try:
>     doc.ensure_timeline("mono")
> except AsrDataError as error:
>     print(error)
> ```

按 ID 读取不存在的数据库记录时使用 Python 标准 `KeyError`。

## normalize_zh {id="normalize-zh",tag="function"}

使用内嵌的中文 TN 规则，把书写形式转换为适合 ASR 比较的口语形式。FST 资源随
`asr-data` 一起提供，不需要调用外部服务。

> #### Example
>
> ```python
> from asr_data import normalize_zh
>
> text = normalize_zh("2024年")
> print(text)  # 二零二四年
> ```

| Name        | Description                   |
| ----------- | ----------------------------- |
| `text`      | 需要归一化的中文文本。~~str~~ |
| **RETURNS** | 归一化后的文本。~~str~~       |

处理失败时抛出 [`AsrDataError`](#asr-data-error)。
