## AudioSource {id="audio-source",tag="class"}

描述尚未解码的音频来源。来源可以是文件路径、URL、编码字节、base64 字符串或 PCM
S16LE 字节。`AudioSource` 可以加载为完整 [`Audio`](#audio)，也可以流式产生
[`AudioChunk`](#audio-chunk)。

## AudioSource.from_path {id="audio-source-from-path",tag="class method"}

从本地音频文件创建来源。文件会在调用 `load()` 或 `stream()` 时解码。

> #### Example
>
> ```python
> from pathlib import Path
> from tempfile import TemporaryDirectory
> from urllib.request import urlretrieve
>
> from asr_data import AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
>
> with TemporaryDirectory() as directory:
>     path = Path(directory) / "example.wav"
>     urlretrieve(URL, path)
>     source = AudioSource.from_path(str(path))
>     audio = source.load()
>     print(audio.duration_ms)
> ```

| Name        | Description                   |
| ----------- | ----------------------------- |
| `path`      | 本地音频文件路径。~~str~~     |
| **RETURNS** | 新的音频来源。~~AudioSource~~ |

## AudioSource.from_pcm {id="audio-source-from-pcm",tag="class method"}

从 PCM S16LE 字节创建来源。

> #### Example
>
> ```python
> import numpy as np
>
> from asr_data import Audio, AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> audio = Audio.from_url(URL).to_mono().resample(16_000)
> pcm_bytes = (
>     np.clip(audio.samples, -1.0, 1.0) * 32_767
> ).astype("<i2").tobytes()
>
> source = AudioSource.from_pcm(
>     pcm_bytes,
>     sample_rate=16_000,
>     channels=1,
> )
> decoded = source.load()
> print(decoded.sample_rate, decoded.channels)
> ```

| Name          | Description                   |
| ------------- | ----------------------------- |
| `data`        | PCM S16LE 字节。~~bytes~~     |
| `sample_rate` | 每秒采样数。~~int~~           |
| `channels`    | 声道数，默认为 `1`。~~int~~   |
| **RETURNS**   | 新的音频来源。~~AudioSource~~ |

## AudioDoc {id="audio-doc",tag="class"}

一条可持久化的音频记录，包含稳定 ID、[`AudioSource`](#audio-source)、
[`AudioInfo`](#audio-info)、业务元数据和各声道的 [`Timeline`](#timeline)。解码后的 [`Audio`](#audio) 和
[`AudioChunk`](#audio-chunk) 不会直接存入 `AudioDoc`。

## AudioDoc.\_\_init\_\_ {id="audio-doc-init",tag="method"}

创建一条音频记录。构造时会探测来源但不解码样本，保存 `audio_info`，并自动创建
全部原始声道的 timeline。

> #### Example
>
> ```python
> from asr_data import AudioDoc, AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> doc = AudioDoc(
>     AudioSource.from_url(URL),
>     id="call-001",
> )
> print(doc.id, doc.source.url)
> print(doc.audio_info.duration_ms, doc.timelines.keys())
> ```

| Name     | Description                                    |
| -------- | ---------------------------------------------- |
| `source` | 音频来源。~~AudioSource~~                      |
| `id`     | 稳定业务 ID；省略时自动生成。~~Optional[str]~~ |

单声道创建 `mono`；双声道创建 `left` 和 `right`；更多声道继续以 `2`、`3` 等
编号。每条 timeline 的整数毫秒时长由 `audio_info` 向上取整得到。异步程序可以使用
`await AudioDoc.afrom_source(source, id=...)`。

## Timeline {id="timeline",tag="class"}

某个声道上的时间轴。它将参考真值与模型预测分成两个独立集合。

| Name          | Description                                         |
| ------------- | --------------------------------------------------- |
| `id`          | Timeline 唯一 ID。~~str~~                           |
| `audio_id`    | 所属 AudioDoc ID，可修改。~~str~~                   |
| `duration_ms` | Timeline 总时长。~~int~~                            |
| `reference`   | 不带 source 的参考标注。~~ReferenceAnnotations~~    |
| `prediction`  | 必须带 source 的预测标注。~~PredictionAnnotations~~ |

所有 `add_*()` 都立即校验 `0 <= start_ms <= end_ms <= timeline.duration_ms`。内
容完全相同的重复添加会返回已有 annotation，不会插入第二条记录。

## Timeline.eval {id="timeline-eval",tag="method"}

使用 reference 评估 prediction。不传 source 时自动评测全部可用来源；也可以为转写
和 Speech 分别指定一个或多个来源。转写默认先执行内嵌中文 TN，再计算 CER。

> #### Example
>
> ```python
> from asr_data import AudioDoc, AudioSource
> from asr_data.annotation import Transcription
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> doc = AudioDoc(AudioSource.from_url(URL))
> timeline = doc.timeline("mono")
> text = "甚至出现交易几乎停滞的情况"
>
> timeline.reference.add_speech(0, timeline.duration_ms)
> timeline.reference.add_transcription(
>     0, timeline.duration_ms, Transcription(text, language="zh")
> )
> timeline.prediction.add_speech(
>     0, timeline.duration_ms, source="silero-vad"
> )
> timeline.prediction.add_transcription(
>     0,
>     timeline.duration_ms,
>     Transcription(text, language="zh"),
>     source="qwen-asr",
> )
>
> result = timeline.eval()
> print(
>     result.transcription["qwen-asr"].cer,
>     result.speech["silero-vad"].f1,
> )
> ```

| Name            | Description                                                   |
| --------------- | ------------------------------------------------------------- |
| _keyword-only_  |                                                               |
| `transcription` | 转写来源或来源列表；省略时自动发现。~~Optional[str \| List[str]]~~ |
| `speech`        | Speech 来源或来源列表；省略时自动发现。~~Optional[str \| List[str]]~~ |
| `normalize`     | 是否执行内嵌中文 TN，默认为 `True`。~~bool~~                  |
| **RETURNS**     | 按任务和 source 分组的结果。~~TimelineEvaluation~~            |

不传两个 source 参数时自动评测全部具有 reference 的任务。只传一个参数时只评测该
任务。显式指定的 reference 或 prediction 不存在，以及完全没有可评测内容时，抛出
[`AsrDataError`](#asr-data-error)。

## AudioDB.query {id="audio-db-query",tag="method"}

按 ID 游标、时长、数据库自动维护的创建/修改时间和元数据筛选记录。

> #### Example
>
> ```python
> from datetime import datetime, timedelta, timezone
> from pathlib import Path
> from tempfile import TemporaryDirectory
>
> from asr_data import AudioDB, AudioDoc, AudioSource
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
>
> with TemporaryDirectory() as directory:
>     path = Path(directory) / "dataset.db"
>     db = AudioDB.create(str(path))
>     doc = AudioDoc(AudioSource.from_url(URL), id="call-001")
>     doc.metadata["split"] = "train"
>     db.insert(doc)
>
>     now = datetime.now(timezone.utc)
>     page = db.query(
>         limit=100,
>         created_from=now - timedelta(days=7),
>         updated_from=now - timedelta(hours=24),
>         metadata={"split": "train"},
>     )
>     print([item.id for item in page])
> ```

| Name              | Description                                          |
| ----------------- | ---------------------------------------------------- |
| `limit`           | 最大返回数，默认为 `100`，上限 `10_000`。~~int~~     |
| _keyword-only_    |                                                      |
| `after`           | 上一页最后一条 audio ID。~~Optional[str]~~           |
| `min_duration_ms` | 可选最短时长。~~Optional[int]~~                      |
| `max_duration_ms` | 可选最长时长。~~Optional[int]~~                      |
| `created_from`    | 创建时间下界，包含。~~Optional[datetime]~~           |
| `created_until`   | 创建时间上界，不包含。~~Optional[datetime]~~         |
| `updated_from`    | 修改时间下界，包含。~~Optional[datetime]~~           |
| `updated_until`   | 修改时间上界，不包含。~~Optional[datetime]~~         |
| `metadata`        | 精确匹配的 JSON 元数据。~~Optional[Dict[str, Any]]~~ |
| **RETURNS**       | 按 audio ID 排序的记录。~~List[AudioDoc]~~           |

时间参数必须是带时区的 `datetime`，并使用 `from <= timestamp < until` 半开区间。
创建和修改时间由 `AudioDB` 自动维护，不会出现在返回的 `AudioDoc` 中。

## AudioDB.eval {id="audio-db-eval",tag="method"}

评测数据库中全部匹配记录。内部通过 AudioDoc ID 游标分页，`batch_size` 只影响每批
读取数量，不限制最终数据集大小。

> #### Example
>
> ```python
> from tempfile import TemporaryDirectory
>
> from asr_data import AudioDB, AudioDoc, AudioSource
> from asr_data.annotation import Transcription
>
> URL = "https://deepasset.oss-cn-beijing.aliyuncs.com/example.wav"
> text = "甚至出现交易几乎停滞的情况"
>
> with TemporaryDirectory() as directory:
>     db = AudioDB.create(f"{directory}/evaluation.db")
>     doc = AudioDoc(AudioSource.from_url(URL), id="sample-1")
>     timeline = doc.timeline("mono")
>     timeline.reference.add_transcription(
>         0, timeline.duration_ms, Transcription(text)
>     )
>     timeline.prediction.add_transcription(
>         0, timeline.duration_ms, Transcription(text), source="qwen-asr"
>     )
>     doc.metadata["split"] = "test"
>     db.insert(doc)
>
>     result = db.eval(
>         transcription="qwen-asr",
>         metadata={"split": "test"},
>         batch_size=100,
>     )
>     print(result.transcription["qwen-asr"].cer)
>     print(result.transcription["qwen-asr"].coverage)
> ```

| Name              | Description                                                   |
| ----------------- | ------------------------------------------------------------- |
| _keyword-only_    |                                                               |
| `transcription`   | 转写来源或来源列表；省略时自动发现。~~Optional[str \| List[str]]~~ |
| `speech`          | Speech 来源或来源列表；省略时自动发现。~~Optional[str \| List[str]]~~ |
| `normalize`       | 是否执行中文 TN，默认为 `True`。~~bool~~                      |
| `batch_size`      | 每次读取的文档数，默认为 `100`。~~int~~                       |
| `after`           | 可选起始游标。~~Optional[str]~~                               |
| `min_duration_ms` | 可选最短时长。~~Optional[int]~~                               |
| `max_duration_ms` | 可选最长时长。~~Optional[int]~~                               |
| `created_from`    | 创建时间下界。~~Optional[datetime]~~                          |
| `created_until`   | 创建时间上界。~~Optional[datetime]~~                          |
| `updated_from`    | 修改时间下界。~~Optional[datetime]~~                          |
| `updated_until`   | 修改时间上界。~~Optional[datetime]~~                          |
| `metadata`        | 精确匹配的 JSON 元数据。~~Optional[Dict[str, Any]]~~          |
| **RETURNS**       | 数据集级聚合结果。~~DatasetEvaluation~~                       |

## AnnotationKind {id="annotation-kind",tag="type alias"}

描述 annotation payload 的类型。

| Value              | Description      |
| ------------------ | ---------------- |
| `"speech"`         | 语音活动范围。   |
| `"token"`          | 细粒度文本单元。 |
| `"transcription"`  | 完整转写。       |
| `"sentence"`       | 句子级文本片段。 |
| `"speaker"`        | 说话人发话。     |
| `"language"`       | 语言信息。       |
| `"acoustic_event"` | 非语音声学事件。 |

`AnnotationSourceKind` 和 silence kind 已删除。

Python 公开的添加方法只创建 `speech`、`transcription` 和 `speaker`。`token` 嵌套
在 [`Transcription`](#transcription) 中；`sentence`、`language` 和
`acoustic_event` 可能出现在从 Rust 或历史数据读取的记录中，但没有对应的 Python
`add_*` 方法。

## AnnotationPayload {id="annotation-payload-alias",tag="type alias"}

`Annotation.payload` 的返回类型。Python 公开的主要 payload 类型统一位于
`asr_data.annotation` 模块。

```python
AnnotationPayload = Speaker | Token | Transcription | dict[str, object] | None
```

`Speech` 的 payload 为 `None`。`Sentence`、`Language` 和 `AcousticEvent` 暂无
Python payload 类，从数据库或 Rust 数据读取时以字典形式返回。
