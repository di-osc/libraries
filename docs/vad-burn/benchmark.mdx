## 测试环境 {id="environment"}

- 音频：`assets/vad_example.wav`, 16 kHz mono PCM, 70.47 s
- 构建：`--release`
- FSMN 模型：`iic/speech_fsmn_vad_zh-cn-16k-common-pytorch@master`
- FireRedVAD 模型：`xukaituo/FireRedVAD@master`

RTF 和 RTFX 会受机器、系统负载、首次 kernel 编译、缓存状态影响。建议在目标机器上
用相同命令重复测试。

## FSMN Flex CPU {id="fsmn-flex"}

```bash
cargo run --release --example bench_fsmn_vad -- \
  --backend flex \
  --audio assets/vad_example.wav \
  --warmup 2 \
  --repeat 10 \
  --stream-chunk-ms 600
```

| 模式 | 平均耗时 | RTF | RTFX |
| --- | ---: | ---: | ---: |
| 离线整段 | 73.631 ms | 0.001045 | 957.08x |
| 流式 600 ms chunk | 198.425 ms | 0.002816 | 355.15x |

## FSMN Metal {id="fsmn-metal"}

```bash
cargo run --release --features metal --example bench_fsmn_vad -- \
  --backend metal \
  --audio assets/vad_example.wav \
  --warmup 2 \
  --repeat 10 \
  --stream-chunk-ms 600
```

本机测试结果：

| 模式 | 平均耗时 | RTF | RTFX |
| --- | ---: | ---: | ---: |
| Metal 离线整段 | 17.823 ms | 0.000253 | 3954.00x |
| Metal 流式 600 ms chunk | 337.176 ms | 0.004785 | 209.00x |

Metal 后端由 Burn WGPU/CubeCL 栈实现。首次 release 构建和首次运行可能包含较长编译
与初始化成本；上表统计的是 warmup 后的 repeat 平均值。

## FireRedVAD {id="firered"}

```bash
cargo run --release --example bench_firered_vad -- \
  --audio assets/vad_example.wav \
  --warmup 2 \
  --repeat 10 \
  --stream-chunk-ms 600
```

| 模式 | 平均耗时 | RTF | RTFX |
| --- | ---: | ---: | ---: |
| 离线 VAD | 96.833 ms | 0.001374 | 727.75x |
| Stream-VAD 600 ms chunk | 176.427 ms | 0.002504 | 399.43x |

## 指标解释 {id="metrics"}

- **RTF**：real-time factor，计算方式是 `处理耗时 / 音频时长`。越小越快。
- **RTFX**：实时倍速，计算方式是 `1 / RTF` 或 `音频时长 / 处理耗时`。越大越快。
- **离线整段**：一次调用 `detect()` 处理完整音频。
- **流式**：按 `--stream-chunk-ms` 切块，多次 `push()` 后调用 `finish()`。
