## 安装 {id="installation"}

```bash
cargo add vad-burn
```

启用 Apple Metal 后端：

```toml
[dependencies]
vad-burn = { version = "0.1.3", features = ["metal"] }
```

## 输入音频 {id="waveform"}

Rust API 使用 [`Waveform`](/vad-burn/api#waveform)。输入必须是 16 kHz 单声道，
样本为归一化 `f32` PCM：

```rust
use vad_burn::Waveform;

let waveform = Waveform::new(samples, 16_000);
println!("{:.3} seconds", waveform.duration_seconds());
```

如果上游音频是 WAV、MP3 或多声道，需要先用自己的音频解码链路转换为
16 kHz mono float PCM，再传给 `vad-burn`。

## FSMN VAD {id="fsmn"}

FSMN 默认模型来自 ModelScope
`iic/speech_fsmn_vad_zh-cn-16k-common-pytorch@master`：

```rust
use vad_burn::{FsmnVadModel, VadOptions, Waveform};

fn main() -> anyhow::Result<()> {
    let model = FsmnVadModel::from_modelscope()?;
    let waveform = Waveform::new(samples, 16_000);
    let segments = model.detect(&waveform, &VadOptions::default())?;
    println!("segments={}", segments.len());
    Ok(())
}
```

本地加载需要目录中包含 `model.pt` 和 `am.mvn`：

```rust
let model = FsmnVadModel::from_pretrained("/path/to/fsmn-vad")?;
```

需要性能诊断时使用 `detect_with_timing()`：

```rust
let detection = model.detect_with_timing(&waveform, &VadOptions::default())?;
println!("forward {:.3} ms", detection.timing.forward_seconds * 1000.0);
println!("frames {}", detection.frame_scores.len());
```

## FireRedVAD {id="firered"}

FireRedVAD 默认模型来自 ModelScope `xukaituo/FireRedVAD@master`。官方仓库包含离线
`VAD` 权重和流式 `Stream-VAD` 权重；`from_modelscope()` 会同时加载两者：

```rust
use vad_burn::{FireRedVadModel, VadOptions, Waveform};

let model = FireRedVadModel::from_modelscope()?;
let waveform = Waveform::new(samples, 16_000);
let segments = model.detect(&waveform, &VadOptions::default())?;
```

本地目录可以是 ModelScope 快照根目录，也可以包含 FireRed 官方权重布局。

## 流式检测 {id="streaming"}

stream 是有状态对象；每条音频流创建一个 stream，按顺序调用 `push()`，最后调用
`finish()`：

```rust
let mut stream = model.new_stream(VadOptions::default());
let mut all_segments = Vec::new();

for chunk in chunks {
    all_segments.extend(stream.push(chunk, 16_000)?);
}

all_segments.extend(stream.finish()?);
```

`push()` 只返回当前已经稳定闭合的片段。`finish()` 会处理最后一块待确认音频并重置
stream 状态。

## Metal 后端 {id="metal"}

FSMN VAD 支持 Burn Metal 后端：

```rust
use burn::backend::{Metal, wgpu::WgpuDevice};
use vad_burn::{FsmnVadModel, VadOptions, Waveform};

let model = FsmnVadModel::<Metal>::from_pretrained_on_device(
    "/path/to/fsmn-vad",
    WgpuDevice::default(),
)?;
let waveform = Waveform::new(samples, 16_000);
let segments = model.detect(&waveform, &VadOptions::default())?;
```

`FsmnVadModel` 的默认类型参数仍是 Flex CPU 后端；因此现有代码不需要修改。Metal
只在 Rust API 暴露，Python wheel 当前使用默认 CPU 后端。
