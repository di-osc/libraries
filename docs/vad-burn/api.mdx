本页只列出 Rust API。Python 类和调用示例见
[Python API 参考](/vad-burn/api-python)。除特别说明外，方法返回
`anyhow::Result<T>`。两个模型都只接受 16 kHz 单声道、归一化 `f32` PCM。

## VadOptions {id="vad-options",tag="struct"}

控制离线和流式 VAD 的分段行为。所有字段都公开。

| Field            | Type  | Default | Description                                  |
| ---------------- | ----- | ------: | -------------------------------------------- |
| `threshold`      | `f32` | `0.6`   | 语音判定阈值；FireRed 中非正值使用 `0.4`。   |
| `min_speech_ms`  | `u64` | `250`   | 丢弃短于该时长的语音片段。                   |
| `min_silence_ms` | `u64` | `500`   | 闭合语音片段所需的静音时长。                 |
| `max_segment_ms` | `u64` | `30000` | 最长语音片段；`0` 的行为因模型而异，见下文。 |
| `pad_ms`         | `u64` | `0`     | 保留字段，当前用于 API 兼容。                |

> #### Example
>
> ```rust
> use vad_burn::VadOptions;
>
> let options = VadOptions {
>     threshold: 0.55,
>     min_silence_ms: 300,
>     max_segment_ms: 15_000,
>     ..VadOptions::default()
> };
> ```

`max_segment_ms=0` 会禁用 FSMN 的最长时长拆分；FireRed 则回退到其内部默认值
20,000 ms。需要统一限制时，应显式传入正值。

## DurationMs、TimeRange 与 VadSegment {id="vad-segment",tag="structs"}

`DurationMs(pub u64)` 表示毫秒值，`TimeRange` 和 `VadSegment` 的字段也都公开：

> #### Example
>
> ```rust
> use vad_burn::{DurationMs, TimeRange, VadSegment};
>
> let segment = VadSegment {
>     range: TimeRange::new(DurationMs(120), DurationMs(980)),
>     probability: 0.82,
> };
> println!(
>     "{}..{} ms, score={:.2}",
>     segment.range.start.0,
>     segment.range.end.0,
>     segment.probability
> );
> ```

## Waveform {id="waveform",tag="struct"}

Rust API 的音频输入类型。

| Field         | Type       | Description                              |
| ------------- | ---------- | ---------------------------------------- |
| `samples`     | `Vec<f32>` | 交错排列的归一化浮点 PCM 样本。         |
| `sample_rate` | `u32`      | 采样率；VAD 推理要求 `16000`。           |
| `channels`    | `u16`      | 声道数；VAD 推理要求 `1`。               |

| Method                                           | Returns    | Description                |
| ------------------------------------------------ | ---------- | -------------------------- |
| `Waveform::new(samples, sample_rate)`             | `Waveform` | 构造单声道波形。           |
| `Waveform::new_with_channels(samples, rate, ch)`  | `Waveform` | 构造指定声道数的波形。     |
| `duration_seconds()`                              | `f64`      | 返回音频时长，单位秒。     |
| `duration_ms()`                                   | `f64`      | 返回音频时长，单位毫秒。   |
| `slice_ms(start_ms, end_ms)`                      | `Waveform` | 按毫秒复制并截取波形。     |

> #### Example
>
> ```rust
> use vad_burn::Waveform;
>
> // 实际使用时替换为解码后的 16 kHz mono float PCM。
> let waveform = Waveform::new(vec![0.0_f32; 32_000], 16_000);
> let first_second = waveform.slice_ms(0, 1_000);
> println!("duration={:.3}s", first_second.duration_seconds());
> ```

## FsmnVadModel {id="fsmn-vad-model",tag="class"}

FSMN VAD 模型，使用 Burn Flex CPU 后端。

| Method                                                  | Description                                      |
| ------------------------------------------------------- | ------------------------------------------------ |
| `from_pretrained(model_dir)`                            | 加载含 `model.pt` 和 `am.mvn` 的本地目录。       |
| `from_modelscope()`                                     | 下载并加载默认 ModelScope 模型。                 |
| `from_modelscope_revision(repo_id, revision)`           | 同步加载指定仓库和 revision。                    |
| `from_modelscope_async()`                               | 默认模型的异步加载入口。                         |
| `from_modelscope_revision_async(repo_id, revision)`     | 指定仓库和 revision 的异步加载入口。             |
| `model_dir()`                                           | 返回已加载模型目录。                             |
| `forward_frame_scores(feats)`                           | 对 `FeatureTensor` 执行前向计算。                |
| `forward_frame_scores_with_timing(feats)`               | 返回帧分数和 `FsmnForwardTiming`。               |
| `detect(waveform, options)`                             | 离线检测并返回 segments。                        |
| `detect_with_timing(waveform, options)`                 | 返回 segments、帧分数和计时。                    |
| `new_stream(options)`                                   | 创建独立的有状态流式检测器。                     |

加载默认模型并执行整段检测：

> #### Example
>
> ```rust
> use vad_burn::{FsmnVadModel, VadOptions, Waveform};
>
> fn main() -> anyhow::Result<()> {
>     let waveform = Waveform::new(vec![0.0_f32; 16_000], 16_000);
>     let model = FsmnVadModel::from_modelscope()?;
>     let segments = model.detect(&waveform, &VadOptions::default())?;
>
>     for segment in segments {
>         println!(
>             "{}..{} ms, score={:.3}",
>             segment.range.start.0,
>             segment.range.end.0,
>             segment.probability
>         );
>     }
>     Ok(())
> }
> ```

## FSMN 异步加载 {id="fsmn-async",tag="class methods"}

异步运行时中应使用异步加载入口，避免创建嵌套 runtime：

> #### Example
>
> ```rust
> use vad_burn::FsmnVadModel;
>
> let model = FsmnVadModel::from_modelscope_revision_async(
>     "iic/speech_fsmn_vad_zh-cn-16k-common-pytorch",
>     "master",
> )
> .await?;
> ```

## FsmnVadStream {id="fsmn-vad-stream",tag="class"}

| Method                      | Returns                   | Description                              |
| --------------------------- | ------------------------- | ---------------------------------------- |
| `push(samples, sample_rate)` | `Result<Vec<VadSegment>>` | 推入一个 chunk，返回已稳定闭合的片段。   |
| `finish()`                  | `Result<Vec<VadSegment>>` | 刷出剩余片段并重置状态。                 |
| `reset()`                   | `()`                      | 清空特征、缓存、分数和后处理状态。       |
| `frame_scores()`            | `&[Vec<f32>]`             | 读取当前累计帧级 posterior。             |
| `options()`                 | `&VadOptions`             | 读取创建 stream 时使用的选项。           |

下面按 500 ms 切块。每条并行音频流应各自调用一次 `new_stream()`：

> #### Example
>
> ```rust
> let mut stream = model.new_stream(VadOptions::default());
> let mut segments = Vec::new();
>
> for chunk in waveform.samples.chunks(8_000) {
>     segments.extend(stream.push(chunk, waveform.sample_rate)?);
> }
> println!("frames before finish={}", stream.frame_scores().len());
> segments.extend(stream.finish()?);
> ```

## FSMN 结果与计时 {id="fsmn-results"}

`FsmnVadDetection` 的字段：

| Field          | Type                  | Description          |
| -------------- | --------------------- | -------------------- |
| `segments`     | `Vec<VadSegment>`     | 检测到的语音片段。   |
| `frame_scores` | `Vec<Vec<f32>>`       | 每帧模型 posterior。 |
| `timing`       | `FsmnVadTiming`       | 推理阶段计时。       |

`FsmnVadTiming` 公开 `frontend_seconds`、`forward_seconds`、
`segmenter_seconds` 和 `forward_ops: FsmnForwardTiming`。
`FsmnForwardTiming` 继续细分输入/输出 tensor、linear、四个 block 的 memory 和
affine、softmax 等耗时。

> #### Example
>
> ```rust
> let detection = model.detect_with_timing(&waveform, &VadOptions::default())?;
> println!("segments={}", detection.segments.len());
> println!("frames={}", detection.frame_scores.len());
> println!("forward_ms={:.3}", detection.timing.forward_seconds * 1000.0);
> println!(
>     "softmax_ms={:.3}",
>     detection.timing.forward_ops.softmax_seconds * 1000.0
> );
> ```

## FireRedVadModel {id="firered-vad-model",tag="class"}

官方仓库根目录包含 `VAD/` 和 `Stream-VAD/`；离线检测和流式检测分别使用对应权重。
为兼容旧布局，也可以传入一个同时用于两种模式、含 `model.pth.tar` 和 `cmvn.ark`
的目录。

| Method                                                  | Description                                  |
| ------------------------------------------------------- | -------------------------------------------- |
| `from_pretrained(model_dir)`                            | 加载本地 FireRedVAD 模型。                   |
| `from_modelscope()`                                     | 下载并加载默认 ModelScope 模型。             |
| `from_modelscope_revision(repo_id, revision)`           | 同步加载指定仓库和 revision。                |
| `from_modelscope_async()`                               | 默认模型的异步加载入口。                     |
| `from_modelscope_revision_async(repo_id, revision)`     | 指定仓库和 revision 的异步加载入口。         |
| `model_dir()`                                           | 返回传入的模型根目录。                       |
| `offline_model_dir()`                                   | 返回离线权重目录。                           |
| `stream_model_dir()`                                    | 返回流式权重目录。                           |
| `detect(waveform, options)`                             | 离线检测并返回 segments。                    |
| `detect_with_timing(waveform, options)`                 | 返回 segments、帧概率和计时。                |
| `new_stream(options)`                                   | 创建使用流式权重的检测器。                   |

> #### Example
>
> ```rust
> use vad_burn::{FireRedVadModel, VadOptions, Waveform};
>
> fn main() -> anyhow::Result<()> {
>     let waveform = Waveform::new(vec![0.0_f32; 16_000], 16_000);
>     let model = FireRedVadModel::from_modelscope()?;
>     let segments = model.detect(&waveform, &VadOptions::default())?;
>
>     for segment in segments {
>         println!("{}..{} ms", segment.range.start.0, segment.range.end.0);
>     }
>     Ok(())
> }
> ```

## FireRedVadStream {id="firered-vad-stream",tag="class"}

| Method                      | Returns                   | Description                              |
| --------------------------- | ------------------------- | ---------------------------------------- |
| `push(samples, sample_rate)` | `Result<Vec<VadSegment>>` | 推入一个 chunk，返回已稳定闭合的片段。   |
| `finish()`                  | `Result<Vec<VadSegment>>` | 刷出剩余片段并重置状态。                 |
| `reset()`                   | `()`                      | 清空前端、FSMN 缓存、分数和后处理状态。  |
| `frame_scores()`            | `&[f32]`                  | 读取当前累计语音概率。                   |
| `options()`                 | `&VadOptions`             | 读取创建 stream 时使用的选项。           |

> #### Example
>
> ```rust
> let mut stream = model.new_stream(VadOptions::default());
> let mut segments = Vec::new();
>
> for chunk in waveform.samples.chunks(8_000) {
>     segments.extend(stream.push(chunk, waveform.sample_rate)?);
> }
> println!("frames before finish={}", stream.frame_scores().len());
> segments.extend(stream.finish()?);
> ```

`finish()` 返回最后的片段后会重置 stream，因此要在调用前读取累计
`frame_scores()`。

## FireRed 结果与计时 {id="firered-results"}

`FireRedVadDetection` 包含 `segments: Vec<VadSegment>`、
`frame_scores: Vec<f32>` 和 `timing: FireRedVadTiming`。
`FireRedVadTiming` 公开 `frontend_seconds`、`forward_seconds`、
`postprocess_seconds` 和处理的声学帧数 `frames`。

> #### Example
>
> ```rust
> let detection = model.detect_with_timing(&waveform, &VadOptions::default())?;
> println!("segments={}", detection.segments.len());
> println!("frames={}", detection.timing.frames);
> println!("forward_ms={:.3}", detection.timing.forward_seconds * 1000.0);
> println!(
>     "postprocess_ms={:.3}",
>     detection.timing.postprocess_seconds * 1000.0
> );
> ```

## 常量与类型别名 {id="rust-exports"}

crate 根和对应模块公开以下模型定位常量：

| Constant                              | Value                                                  |
| ------------------------------------- | ------------------------------------------------------ |
| `DEFAULT_MODELSCOPE_REPO_ID`          | `iic/speech_fsmn_vad_zh-cn-16k-common-pytorch`         |
| `DEFAULT_MODELSCOPE_REVISION`         | `master`                                               |
| `DEFAULT_FIRERED_MODELSCOPE_REPO_ID`  | `xukaituo/FireRedVAD`                                  |
| `DEFAULT_FIRERED_MODELSCOPE_REVISION` | `master`                                               |

FSMN 还公开 `FeatureTensor = burn::tensor::Tensor<burn::backend::Flex, 2>`，供
`forward_frame_scores*()` 使用。
